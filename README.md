# react-boiler

boilerplate for a react project with express server and also gulp.

### global packages

```
npm install -g gulp-cli
npm install -g nodemon
npm install -g webpack
```

### dependencies

```
npm install
```

### build it

```
npm run build
```

### serve it

``` 
npm run server
```

### dev-build with watchers for files and server

```
npm run dev
```
